# Activer le Mature Content

## Qu'est ce que le Mature Content ?

Le Mature Content est un DLC permettant au contenu mature créé et publié par les utilisateurs de la communauté d'être vu, affiché ou accessible dans le jeu.

:::caution Attention !

Le Mature Content peut être bloqué par défaut dans certains pays.

Vous devez aussi être majeur dans votre pays afin de pouvoir activer ce DLC

:::

## Activer le contenu à caractère sexuel sur Steam

En premier temps, rendez vous dans vos paramètres de compte en cliquant sur votre icone de compte, puis sur **Détails du compte**
<img src={useBaseUrl('/img/tutos/mature_content/1.png')} alt="1" width="300" />

Une fois dans les détails de votre compte, rendez vous dans les **Paramètres** à l'aide du bouton du même nom sur la barre à gauche
<img src={useBaseUrl('/img/tutos/mature_content/2.png')} alt="2" width="300" />

Une fois rendu dans les paramètres, rendez vous dans la première fenêtre puis dans la catégorie **CONTENU POUR ADULTES**. Une fois la catégorie trouvée, cochez la case : **Contenu à caractère sexuel réservé aux adultes**
<img src={useBaseUrl('/img/tutos/mature_content/3.png')} alt="3" width="600" />

Voila ! Le contenu à caractère sexuel est activé sur votre compte Steam, vous pouvez désormais vous rendre sur la page du [DLC Mature Content](https://store.steampowered.com/app/1435530/ChilloutVR__Mature_Content_Access/) afin de l'activer.
